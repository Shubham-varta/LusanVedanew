# Workflow Rules

Always read before coding:

- AGENTS.md
- GOAL.md
- ROADMAP.md
- ARCHITECTURE.md

Follow ROADMAP.md exactly.

Never skip phases.

Implement only one phase.

Never continue automatically.

After completing a phase:

- Verify build
- Verify existing features
- Explain modified files
- Explain commands
- Suggest commit message

Stop.

Wait for approval.

Never commit.

Never push.

Never create unnecessary files.

If documentation conflicts with code, ask the user.
